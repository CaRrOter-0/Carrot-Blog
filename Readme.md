# Carrot-Blog
* The small blog system production
* 个人博客系统网址：http://175.178.159.253/
## 技术栈

前端：
* JS框架：JQuery
* CSS框架：Semantic UI

后端：
* 核心框架：springboot
* JDK：1.8.0
* 持久层：Mybatis
* 模板框架：Thymeleaf
* 分页插件：PageHelper
* 部署环境：腾讯云Centos8

数据库：
* MySQL5.7
